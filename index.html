Introduction

This course will focus on optimization techniqeus for networking systems. In this course, we discuss the following topics:

  • Concept of Network optimization
  • Decision Produre and Optimization Techniques
    • Satisfiability (SAT)
    • Satisfiability Modulo Theory (SMT)
    • Integer Linear Programming (ILP)
    • Constraint Programming(CP)
    • Model-based Learning: STRATEGO in UPPAAL.
  • Optimization Problems and Algorithms for Network Systems
